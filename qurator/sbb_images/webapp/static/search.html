Mensch.Maschine.Kultur(current)
This is an AI-based image search tool which allows you to search for images inside documents from the Digitized Collections of the Berlin State Library (SBB). You can start your search either by uploading an image, or describing the image you are looking for. All the search results will be images. The underlying technology is open source and actively being developed by SBB in the Mensch.Maschine.Kultur project.
